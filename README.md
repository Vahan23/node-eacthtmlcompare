## Requirements
 - MySQL community server
 
 - MongoDB 
 
 - NodeJs



## Installation

1. run  *npm install*  in root directory 

2. run  *npm install*  in client folder 

3. run *node setup.js* in setup folder and follow installation steps



## Starting Development server

run *npm run dev*
